# 🏥 Medicenter - Landing Page Responsiva

Este projeto consiste em uma landing page responsiva inspirada no template **Medicenter**, desenvolvida com **HTML5** e **CSS3**, utilizando **Flexbox** para a organização do layout.

## 📖 Sobre

O objetivo deste projeto foi praticar conceitos de desenvolvimento front-end, principalmente:

- Estruturação semântica em HTML5;
- Layout responsivo utilizando Flexbox;
- Organização de componentes reutilizáveis;
- Estilização com CSS3.

A página simula o site de uma clínica médica contendo diversas seções informativas.

## ✨ Funcionalidades

- Header com logo e menu de navegação;
- Banner principal com chamada de destaque;
- Cards de informações rápidas;
- Seção de últimas notícias;
- Área de departamentos;
- Informações para agendamento;
- Rodapé completo com:
  - Informações da clínica;
  - Últimas postagens;
  - Últimos tweets.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
